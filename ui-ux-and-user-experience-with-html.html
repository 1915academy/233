UI UX and User Experience with HTML

Jan 31, 2023

How good is the user's experience with what the user sees on a website. It will also affect how easily they can use the entire site.

Adding images to the background of parts of a website is often more visible and interesting than changing the background color.

Modern browsers support a variety of image file types such as .jpg, .png, .gif, and .svg.

This article describes how to add photos to your HTML code and how to look better, how to fix them.


Background image :

The first step is to ensure that you make the assets directory (folder) to hold the images you want to use in your project.

For example, we can create an image folder in the project we are working on and add a picture called Sunset.PNG that we want to use.

Background image CSS property allows you to place the image behind any HTML element as you wish.

This can either be the whole page (using body selectors in CSS that targets the  factor in our HTML), or only one standstone of the page specific parts such as the section element in the bottom example.

To add background icons to a section tag in your .CSS file, write the following code:

section {

     background-image: url("images/sunset.png");

        }

Let's discuss what's happening here:


The section explains the tag that you want to add to the picture.

The URL () is used to tell the CSS where our image is located.

Inside the bracket, "Images/Sun Set. PNG" is the way to icon. This allows the browser to know if the URL is to be pulled.

The path to this example is called a relative route, which means it is a local file, which is related to our project and our current working directory and this is no web address. Adding photos We can also use an absolute path, which is a complete web address and begins with the domain URL (http: // www.).

Prices are a good habit, but we can leave them, so the background image: URL (photos/sunset - PNG) does the same thing.

Don't forget to semiculose!

If the picture is smaller than the tag that has a background, it will repeat to fill the tag.

How do we stop it from happening?

The background property is taken in 4 values and we are capable of changing the direction in which the image repeats, or prevents the image from repeating ourselves together.
