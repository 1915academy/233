The common syntax for HTML comment seems like this:

<script> </script>

Comments in HTML begin with " . " which is a period 

Don't forget the amazing mark at the beginning of the tag! But you don't have to add it in the end.

The tag is around any text or other HTML tag that you want to comment on.

Use html comment :

HTML comments do not appear in the browser. This means that no comments will be shown when you add to your HTML source code when the document is presented in a web browser.

It is being said, keep in mind that anyone can see the source code of every website published on the Internet -> Developer -> Watch the Source -and that includes all the comments!

So your comments will appear for others whether you make the HTML document public and they choose to see the source code.

Writing comments is helpful and it is a good process to follow the source code. Comments help you to do and communicate with yourself (and others) about your code and thinking process. It also reminds you that when you come back to a project after not working in a project, what were you thinking.

Comments also help you interact with other developers who are working on the project with you. Your comments can clarify them why you have added some code lines.

Single-Line Comments in HTML

A single-line comment only spans one line. As mentioned earlier, that line will not get displayed in the browser.

Use a single-line comment when you want to explain and clarify the purpose behind the code that follows it or when you want to add reminders to yourself like so:

Single-line comments are also helpful when you want to make clear where a tag ends. This comes in handy in a long and complex HTML document where a lot is going on and you may get confused as to where a closing tag is situated.

Inline Comments in HTML

You can also add comments to the middle of a sentence or code line.

The text will only be commented on within , and the rest of the text inside the tag will not be affected.

Shortcut for Adding HTML Comments

You can use a shortcut to add comments - and you will probably use them too much. Shortcut Command / Mac is for users or for control / Windows and Linux users.

To add a single line comment, the keys shown inside the code editor hold the scroll. Then your entire line will be commented. Keep in mind that since everything will be commented on this line, it only works for one line comments. You will need to manually add in -line comments.

To add multi -line comments, select and highlight all the text or tags you want to comment on and hold the two keys shown before. There will now be a comment on each line you have selected.
